API Basejump: URL Shortener

User stories:
  1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response
  2. When I visit that shortened URL, it will redirect me to my original link.

Example usage:

https://kinley-urlshort.herokuapp.com/new/https://www.google.com
https://kinley-urlshort.herokuapp.com/new/http://freecodecap.com/news

Example creation output

{"original_url":"http://freecodecamp.com/news","short_url":"https://kinley-urlshort.herokuapp.com/4"}

Usage:

https://kinley-urlshort.herokuapp.com/4

Will redirect to:

http://freecodecamp.com/news
